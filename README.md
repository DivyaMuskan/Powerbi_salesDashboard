# Power BI Sales Dashboard

## Project Overview
This Power BI dashboard provides an interactive analysis of sales performance across different regions, products, and customer segments. The dashboard helps identify revenue trends, top-performing products, and key business insights through data visualization.

## Objectives
- Analyze overall sales performance
- Track revenue and profit trends
- Identify top-performing products
- Compare regional sales performance
- Monitor customer purchasing behavior
- Provide interactive business insights

## Tools & Technologies
- Power BI
- Microsoft Excel
- DAX
- Data Modeling
- Data Visualization

## Dashboard Features

### KPI Cards
- Total Sales
- Total Profit
- Total Orders
- Total Customers

### Visualizations
- Sales by Region
- Sales by Product Category
- Revenue Trend Analysis
- Customer Analysis
- Interactive Map Visualization
- Monthly Sales Performance

### Interactive Elements
- Slicers
- Filters
- Drill-down Analysis
- Dynamic Visualizations

## Key Insights
- Identified top-performing regions and products
- Analyzed monthly revenue trends
- Evaluated customer purchasing patterns
- Compared performance across different categories
